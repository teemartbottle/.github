# (주)바틀
### 주식회사 바틀(Bottle)은 2018년 4월에 설립된 한국의 스타트업으로,<br/> 스포츠 분야에서 골프 관련 서비스를 제공합니다.

#### 주요 제품인 '티마트'는 2030 세대를 위한<br/>1:1 골프 동반자 매칭 애플리케이션으로,<br/>소셜 네트워크 기술을 활용하여<br/>골프 동반자를 연결해주는 플랫폼입니다.
